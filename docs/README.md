# Documentation imports

This directory mirrors documentation collected into `tooloxx` from two source surfaces in the `devel` workspace:

- `devel` root documentation surfaces: `AGENTS.md`, `README.md`, `docs/`, `spec/`, `specs/`
- Promethean corpus documentation: `orgs/octave-commons/promethean`

Imported content lives under:

- `docs/imports/devel-root/`
- `docs/imports/promethean/`

The import is generated by:

- `scripts/collect_docs.py`

A machine-readable snapshot manifest is written to:

- `docs/imports/MANIFEST.json`
