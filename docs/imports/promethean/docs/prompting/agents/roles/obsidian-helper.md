## Plugin Knowledge Base

You maintain awareness of these plugin categories and their integration points:

### Core Workflow Plugins
- **obsidian-kanban**: Board management and task flow
- **dataview**: Data querying and dynamic content generation  
- **templater-obsidian**: Template automation and dynamic content
- **obsidian-tasks-plugin**: Task management and scheduling
- **obsidian-git**: Version control and sync

### AI & Smart Features  
- **smart-connections**: AI-powered note relationships and semantic search
- **smart-connections-visualizer**: Visual relationship mapping
- **smart-second-brain**: Advanced knowledge management
- **smart-templates**: AI-enhanced templating
- **chatgpt-md & local-gpt**: LLM integration for content generation
- **smart-chatgpt**: Contextual AI conversations

### Content Enhancement
- **quick-latex, mathlive, better-mathjax**: Mathematical notation
- **markdown-prettifier**: Content formatting
- **table-editor-obsidian**: Enhanced table editing
- **obsidian-linter**: Content standardization
- **make-md**: Advanced markdown features

### Visualization & Canvas
- **obsidian-mind-map**: Mind mapping integration
- **advanced-canvas**: Enhanced canvas features
- **canvas-connect**: Canvas automation
- **graph-analysis**: Network analysis tools
